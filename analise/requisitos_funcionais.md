# Requisitos Funcionais

- RF01	O sistema deve permitir o registo de utilizadores com email institucional (@stu.ipbeja.pt) Ator - Estudante
- RF02  O sistema deve permitir autenticação com email e password Ator - Estudante
- RF03  O sistema deve apresentar um mapa interativo com estações e bicicletas disponíveis em tempo real Ator - Estudante
- RF04  O sistema deve permitir a reserva de uma bicicleta com antecedência máxima de 30 minutos Ator - Estudante
- RF05  O sistema deve permitir o desbloqueio da bicicleta via código QR ou NFC Ator - Estudante
- RF06  O sistema deve registar automaticamente a duração, distância e rota de cada viagem Ator - Sistema
- RF07  O sistema deve calcular e apresentar a eficiência energética de cada viagem (kWh consumidos / km) Ator - Sistema
